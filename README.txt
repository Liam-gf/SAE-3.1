# README Groupe 8

## Sur Linux

### Compilation
Pour compiler le programme, placez-vous dans le dossier `SAE-3.1` puis exécutez :
javac -encoding UTF-8 @compile.list

### Exécution
Pour executer le programme, restez dans le dossier `SAE-3.1` puis exécutez :
java -cp class src.Controleur
